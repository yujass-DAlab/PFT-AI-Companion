## Overview

PFT Prep Coach is an AI-assisted educational application designed to prepare patients before undergoing spirometry test, which is the most demanding part of the Pulmonary Function Testing (PFT).

Many patients experience anxiety because the breathing maneuvers performed during PFT are very different from normal breathing.

This project aims to improve patient understanding, confidence and cooperation before the actual examination.

The application is **NOT** intended to diagnose disease.

Instead, it functions as a virtual preparation coach that complements the guidance provided by respiratory therapists.

---

## Motivation

This project was inspired by a real first-time patient experience.

During Pulmonary Function Testing, the breathing maneuvers required by the testing software were very demanding.

Although the respiratory therapist provided coaching, translating verbal instructions such as

> "Harder"

> "Faster"

> "Keep blowing"

into precise physical actions proved difficult.

This project explores whether better patient preparation may improve confidence, communication and overall testing experience.
